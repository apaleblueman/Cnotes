# Unions

> Unions are a field that assign multiple labels of multiple types to the same memory location. 
> They are similar to structs but their size is as large as largest element in them.

## Syntax
`union unionname{`
`    datatype var1;`
`    datatype var2;`
`    .`
`    .`
`    .`
`    datatype varN;`
`};`
